# 🚀 DevFlow Agent - 开发工作流智能体

**English** | [简体中文](#简体中文)

---

## 🌟 简介

一个专为编程开发打造的智能体工具，用文件夹和Markdown定义开发工作流，直接丢给豆包、Claude、Cursor等AI平台使用。

**专注开发，高效工作！** ✨

### 核心优势
- 📁 **文件夹即智能体**: 用简单的文件夹结构定义完整工作流
- 📝 **Markdown配置**: 无需编程，用YAML配置开发流程
- 🚀 **拖放即用**: 直接把文件夹拖给AI平台就能用
- 🔧 **开发工作流**: 内置16个开发相关工作流
- 🌍 **多平台兼容**: 支持豆包、Claude、Cursor等

---

## 🚀 快速开始

### 第一步：下载模板
```bash
git clone https://github.com/badhope/skills.git
cd skills/example-agents/full-stack-assistant
```

### 第二步：直接使用
把 `full-stack-assistant/` 文件夹拖到您的AI平台即可。

### 第三步：开始工作
用自然语言描述开发需求：
```
"创建一个React Todo应用"
"修复登录页面的Bug"
"审查我的代码质量"
```

---

## 📂 智能体结构

```
full-stack-assistant/
├── agent.yaml          # 智能体配置
├── system-prompt.md    # 角色定义
├── workflow/           # 工作流
│   ├── intent.yaml     # 意图识别
│   ├── stages.yaml     # 流程阶段
│   └── tools.yaml      # 工具列表
├── knowledge/          # 知识库
└── tests/              # 测试用例
```

---

## 📋 开发工作流

| 工作流 | 描述 |
|--------|------|
| `new-project` | 新项目创建 |
| `feature-implementation` | 功能实现 |
| `bug-fixing` | Bug修复 |
| `code-review` | 代码审查 |
| `technical-design` | 技术设计 |
| `deployment` | 部署流程 |
| `security-audit` | 安全审计 |

---

## 🎯 使用示例

```
输入: "创建一个React Todo应用，使用TypeScript"

智能体自动:
1. 识别意图 → new-project
2. 执行工作流 → full-project-workflow
3. 输出: 完整项目结构
```

## 🔌 平台支持

| 平台 | 使用方式 |
|------|----------|
| 🫘 **豆包** | 直接上传文件夹 |
| 🤖 **Claude Desktop** | 添加文件夹路径 |
| ✨ **Cursor** | `@load ./folder` |
| 🌊 **Windsurf** | 添加工作区目录 |
| 🚀 **Trae** | 添加项目目录 |
