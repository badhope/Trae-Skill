## 使用方式

### AI 使用
1. 下载整个仓库
2. 阅读 INDEX.md 了解全局结构
3. 查看对应模块的 registry 定位具体内容
4. 直接引用 skill/prompt/workflow 文件

### 人类使用
1. 直接浏览对应目录
2. 参考 templates 创建新内容
3. 遵循命名规范维护仓库
