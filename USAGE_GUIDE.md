# 📖 Mega-Agent Developer Platform - 使用指南
---
## 🎯 快速开始
### 加载到你的AI平台
| 平台 | 加载方式 |
|------|---------|
| **Trae** | 直接添加为项目目录 |
| **Cursor** | 使用 `@load ./skills` 命令 |
| **Claude Desktop** | 在设置中添加文件夹路径 |
| **Windsurf** | 添加为工作区目录 |
| **Cline** | 配置MCP服务器指向 `skills/mcp` |

---
## 🤖 智能体工作原理
### 完整工作流程
```
用户请求
   │
   ▼
意图分析
   │
   ▼
关键词匹配
   │
   ├─→ 选择专家引擎 (13个可选)
   │   │
   │   ├─→ 前端大师 → 前端开发
   │   ├─→ 后端大师 → 后端开发
   │   ├─→ 数据库专家 → 数据库操作
   │   ├─→ DevOps工程师 → 部署
   │   └─→ ...其他专家...
   │
   └─→ 选择元智能体 (5个可选)
       ├─→ 任务编排器 → 复杂任务分解
       ├─→ 任务规划师 → 详细步骤规划
       └─→ ...其他元智能体...
   │
   ▼
调用MCP工具
   │
   ▼
执行任务
   │
   ▼
返回结果
```

## 💬 使用示例
### 示例1：创建一个Web应用
```
用户: "帮我创建一个React+Express的待办应用"

智能体自动:
1. 选择"全栈开发工程师"
2. 调用frontend-dev-kit + backend-dev-kit工具

```

## 🏆 最佳实践
### 提问技巧
1. **明确目标**: 清楚描述想要实现什么
2. **提供上下文**: 如有现有代码或环境，提供相关信息
3. **使用关键词**: 可以帮助智能体快速选择正确的专家
4. **分步骤**: 复杂任务可以分步骤进行

### 输出质量
- 智能体会优先使用MCP工具执行操作
- 代码会遵循最佳实践和规范
- 重要变更会有注释说明
- 测试用例可能会一起生成
