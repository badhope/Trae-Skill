# AI Skill Repository - Design Specification

## 1. Design Style
- **Aesthetic**: Ultra-Minimalism
- **Key Characteristics**: 
  - Clean, spacious layouts with generous whitespace
  - Focus on content and readability
  - Subtle animations and transitions
  - Professional and elegant appearance
  - Monochromatic with accent colors

## 2. Color Palette
- **Background**: #FFFFFF (Pure White)
- **Primary Text**: #1A1A1A (Near Black)
- **Secondary Text**: #666666 (Medium Gray)
- **Accent Primary**: #2563EB (Blue)
- **Accent Secondary**: #10B981 (Green)
- **Border**: #E5E5E5 (Light Gray)
- **Code Background**: #F8F9FA (Off White)

## 3. Typography
- **Heading Font**: Inter (Sans-serif)
  - H1: 48px / 700
  - H2: 36px / 600
  - H3: 24px / 600
  - H4: 18px / 600
- **Body Font**: Inter (Sans-serif)
  - Regular: 16px / 400
  - Small: 14px / 400
  - Line Height: 1.7

## 4. Iconography
- **Library**: Heroicons (Outline style)
- **Style**: Thin stroke, 1.5px
- **Size**: 24px default, 20px small

## 5. Logo
- **Placement**: Header left, centered on mobile
- **Scaling**: 40px height on desktop, 32px on mobile
- **Format**: SVG with text

## 6. Spacing System
- **XS**: 4px
- **SM**: 8px
- **MD**: 16px
- **LG**: 24px
- **XL**: 32px
- **2XL**: 48px
- **3XL**: 64px

## 7. Animation
- **Duration**: 200ms (fast), 300ms (normal), 500ms (slow)
- **Easing**: cubic-bezier(0.4, 0, 0.2, 1)
- **Types**: Fade, Slide, Scale

## 8. Components
- **Buttons**: Rounded corners (8px), subtle hover shadow
- **Cards**: No border, subtle shadow on hover
- **Inputs**: Underline style, minimal
- **Navigation**: Fixed top, transparent background
