架构设计

模块化架构

清晰的目录结构，便于扩展和维护

skill/
├── .trae/
│   └── skills/           # 77+ 技能模块
│       ├── coding/       # 代码生成
│       ├── debugging/    # 调试修复
│       ├── planning/     # 任务规划
│       ├── security/     # 安全审计
│       └── workflows/    # 工作流
├── prompts/              # 132+ 提示词
│   ├── task/            # 任务提示词
│   ├── meta/            # 元提示词
│   └── output/          # 输出格式
├── docs/                 # 文档
└── examples/             # 示例
