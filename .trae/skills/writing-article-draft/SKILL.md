# Writing Article Draft Skill

A systematic approach to creating well-structured written content.

## When to Use

- User wants to write an article or blog post
- User needs documentation created
- User wants to structure their thoughts into written form
- User needs content expanded from outline

## Article Creation Process

### 1. Define Topic and Purpose

Before writing, clarify:
- What is the main topic?
- Who is the target audience?
- What should the reader take away?
- What tone and style is appropriate?

### 2. Create Outline

Structure the article:

```
1. Introduction
   - Hook/attention grabber
   - Topic introduction
   - Preview of main points

2. Main Body
   - Point 1 with supporting details
   - Point 2 with supporting details
   - Point 3 with supporting details

3. Conclusion
   - Summary of main points
   - Call to action or final thought
```

### 3. Write Content

Follow the outline:
- Use clear, concise language
- Include examples and evidence
- Maintain consistent tone
- Use appropriate formatting

### 4. Review and Refine

After drafting:
- Check for clarity and flow
- Verify all points are covered
- Ensure strong conclusion
- Proofread for errors

## Article Types

| Type | Purpose | Typical Length |
|------|---------|----------------|
| How-to | Teach a process | 1000-2000 words |
| Listicle | Present items | 800-1500 words |
| Opinion | Express viewpoint | 500-1000 words |
| Tutorial | Step-by-step learning | 2000-3000 words |
| News | Report information | 300-800 words |

## Quality Checklist

- [ ] Clear thesis statement
- [ ] Logical structure
- [ ] Supporting evidence
- [ ] Smooth transitions
- [ ] Strong conclusion
- [ ] Proper formatting
- [ ] Error-free grammar
