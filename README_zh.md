# Thesis Specialist

---

## For Chinese Users / 中文用户说明

This project provides a Folder-as-Agent platform for academic thesis writing.

### Quick Start / 快速开始

1. Download this folder
2. Submit to any LLM (Doubao, Claude, GPT, Gemini)
3. Describe your thesis requirements in natural language
4. The agent will execute the complete workflow automatically

### Features / 功能特性

- **8-Phase Execution**: Intent recognition → Expert matching → Task planning → Execution → Tool calling → Result integration → Quality check → Final output

- **8 Expert Engines**: Topic selection, Literature review, Structure planning, Writing, English polishing, Data analysis, Reference management, Format compliance

- **4 Meta Agents**: Coordinator, Task planner, Reviewer, Progress tracker

- **4 Tools**: Literature search, Grammar checker, Data visualizer, Format converter

- **Memory System**: User preferences, conversation history, project state, knowledge base

### Usage Examples / 使用示例

| Task | Example Input |
|------|--------------|
| Topic Selection | "Help me come up with a research topic" |
| Literature Review | "Write a literature review on machine learning" |
| Thesis Planning | "Plan my thesis workflow" |
| Full Thesis | "Complete my entire thesis" |
